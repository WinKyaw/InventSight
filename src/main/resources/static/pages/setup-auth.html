Troubleshooting

  • If you get "Company ID not found", set the Company ID field above
  • If you get "403 Forbidden", check your user role (must be GM+ or have Supply Management permission)
  • If you get "401 Unauthorized", your token may be expired - login again
  • Check browser console (F12) for detailed error messages
